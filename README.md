# Sistema-de-gestion-de-informacion-fitosanitaria-para-viveros-
JANGRO - Sistema de Gestión Fitosanitaria
JANGRO es una aplicación web desarrollada en Oracle APEX diseñada para la digitalización y optimización del registro de aplicaciones fitosanitarias, control de plagas y gestión de inventario de productos químicos en cultivos (específicamente orientado a flores/viveros).

🚀 Características Principales
Registro de Aplicaciones: Control detallado de fechas, horas, productos aplicados y áreas tratadas.

Gestión de Rangos: Capacidad de seleccionar múltiples sectores de cultivo en una sola aplicación.

Seguridad y Roles: Control de acceso mediante esquemas de autorización para perfiles como Administrador, Scouter y Aplicador.

Reportes Interactivos: Visualización de datos en tiempo real con nombres reales de productos y plagas (no solo IDs).

Interfaz Amigable: Diseño adaptativo para uso en campo.

🛠️ Tecnologías Utilizadas
Plataforma: Oracle APEX 23.x / 24.x

Base de Datos: Oracle Database (SQL & PL/SQL)

Seguridad: APEX Authentication & Authorization Schemes

Diseño: Universal Theme
